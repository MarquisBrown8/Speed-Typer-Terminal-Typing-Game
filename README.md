# ⌨️ Speed Typer

A terminal typing speed game to test your WPM and accuracy in real-time.

## 🚀 Features
- 10 random tech-related words
- Countdown before start
- WPM & accuracy score
- Option to replay

## ▶️ How to Play

```bash
pip install -r requirements.txt
python speed_typer.py
